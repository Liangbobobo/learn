github不要上传大文件，否则.git文件过大，影响使用

git add .  向这个本地的代码仓库添加当前目录的所有改动

git add -p  只添加某个文件

git status  查看仓库状态 可以查看改便的文件状态

git commit -m "2021.2.5初始使用git整理的文档"

git push origin master  推送到远程仓库 


重要：
git pull --rebase origin master  将远程库与本地库同步
